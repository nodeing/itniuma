## 学习任务

关于每天的学习任务，我们不做要求，大家根据自己的时间来安排，建议每天安排 2 个小时以上的时间来学习，尤其需要注意的是，学习的过程不能中断，中断以后，大概率会忘记前面所学的知识，想要继续学习又必须重新把前面的知识复习一遍，重复入门，这样学习进度很难推进，中断学习 1-2 周以上，效果就会非常的差，不管你什么原因，最好不要中断学习，实在忙不过来，每天至少要学习半个小时，以免前面的知识搞忘记
