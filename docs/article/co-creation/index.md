## 保障

从我们的收费模式你就应该能看出我的目的，就是想拉一些人一起共同做内容，所以我们的 3 个付费版本里面，共创版是最划算的，我最需要的是你帮我创作课程内容，然后我去卖，所以我对课程版权更看重，你需要跟我签版权协议，到时候产出的课程内容版权归我，归我以后，免费授权给你，这样我们都有权力可以来卖这个课程

在这个共创版本里面，我会提供价值`14999 元/半年`的陪跑服务，这个才是核心，在这个过程中，我会协助你完成课程的创作，手把手带你创作一套课程，给你相关的指导，这会消耗我大量的精力，但能确保你能产出可以卖钱的课程作品，产出的这个作品我说能卖钱，这个口说无凭，得经过市场验证才行，因此就有以下的保障：

```
1.第一套课程作品在一年内没有卖到999元，免费继续辅导做第二套课程

2.第一套课程在第一年内没有卖够999元的部分，由我这边补齐，例如：你第一套课程仅卖100块，那剩下的899由我补给你

3.第二套课程上线后，还是以一年为限，没有卖够999元的，免费补齐999
```

以上这几点是保底机制，即能保障你支付的 1998 元现金，能在 3 年内（一套课程做 1-6 个月，卖一年）全部收回

我们双方需要承担的共同风险就是时间的浪费，我辅导你做课程，浪费了时间，你自己半途而废浪费时间，在签订合同的时候，会约定好双方的权利和义务，谁造成时间的浪费，谁承担责任，但总体不会超过 1998 元

:::danger

注意：如果你在有大厂经验加持、比较出名的开源作品加持的情况下，课程依然卖不出去，那我们将在课程上线一年后，退回你的全部费用，不收你一分钱费用（仅共创版用户享受此政策）
