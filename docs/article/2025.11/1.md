# Google Gemini 3 体验教程

## 需要的工具

:::tip
1.Claude Code
2.CC Switch
3.API 网站：https://zenmux.ai/
:::

## 步骤

第 1 步，先注册代理网站 [https://zenmux.ai/](https://zenmux.ai/),经过多个渠道测试，这个网站对于在国内使用 Gemini 体验更好，速度还行

第 2 步，在网站上创建好 API key

![20251119130047](https://img.bokecity.com/ndedu/20251119130047.png)

在右侧个人中心的下拉菜单里面有 API keys 菜单，进入到里面，点创建按钮即可

![20251119130216](https://img.bokecity.com/ndedu/20251119130216.png)

需要注意图中的地址，等会儿配置的时候需要用到

第 3 步，安装 cc switch，下载地址：[https://github.com/farion1231/cc-switch/releases/tag/v3.7.0](https://github.com/farion1231/cc-switch/releases/tag/v3.7.0)

页面往下拖到底，有下载地址，根据你自己电脑的系统来选择

![20251119131128](https://img.bokecity.com/ndedu/20251119131128.png)

第 4 步，配置 cc switch

先点击添加供应商

![20251119131522](https://img.bokecity.com/ndedu/20251119131522.png)

选择自定义配置

![20251119131604](https://img.bokecity.com/ndedu/20251119131604.png)

以下是我填写的配置，你需要替换成你自己的 api key

![20251119131704](https://img.bokecity.com/ndedu/20251119131704.png)

![20251119131737](https://img.bokecity.com/ndedu/20251119131737.png)

需要注意，我填的是付费版模型，你如果要使用免费模型，模型名称应该填：google/gemini-3-pro-preview-free

:::tip
注意：免费模型我自己体验下来不太行，应该是使用的人太多了，有限制，很容易 429，有条件可以使用付费模型，支持支付宝充值
:::
