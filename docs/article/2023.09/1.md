# 程序员常用英语词汇(1-100)

## 词性缩写

:::tip

vi：缩写表示 "intransitive verb"，即不及物动词。不及物动词是指不需要接受动作的宾语，通常用于描述主语的行为或状态，如 "run"（奔跑）和 "sleep"（睡觉）

vt：缩写表示 "transitive verb"，即及物动词。及物动词是指需要接受动作的宾语的动词，通常用于描述主语对宾语进行的动作，如 "eat"（吃）和 "write"（写）

n：缩写表示 "noun"，即名词。名词是用来表示人、事物、地点或概念的词语，如 "cat"（猫）和 "book"（书）

adj：缩写表示 "adjective"，即形容词。形容词是用来描述名词的特征或性质的词语，如 "happy"（快乐的）和 "beautiful"（美丽的）

conj：缩写表示 "conjunction"，即连词。连词是用来连接词语、短语或句子的词语，用于构建复合句或表达逻辑关系，如 "and"（和）和 "but"（但是）

adv：缩写表示 "adverb"，即副词。副词用于描述动词、形容词、其他副词或整个句子的方式、程度、时间或地点，如 "quickly"（迅速地）和 "very"（非常）

prep：缩写表示 "preposition"，即介词。介词用于建立名词与其他词语之间的关系，通常描述位置、方向或时间关系，如 "in"（在）和 "on"（在...上）

pron：缩写表示 "pronoun"，即代词。代词用于替代名词，以避免重复使用相同的名词，如 "he"（他）和 "she"（她）

num：缩写表示 "numeral"，即数词。数词用于表示数字，如 "one"（一）和 "twenty"（二十）

art：缩写表示 "article"，即冠词。冠词用于指定名词的特定性或泛指性，包括 "a"（不定冠词）和 "the"（定冠词）

int：缩写表示 "interjection"，即感叹词。感叹词用于表示强烈的情感或突然的表达，如 "Wow!"（哇！）和 "Ouch!"（哎呀！）

pro：缩写表示 "pronoun"，即代词。这个缩写也可以用来表示 "professional"（专业的）或其他上下文相关的意思，具体含义取决于上下文

:::
