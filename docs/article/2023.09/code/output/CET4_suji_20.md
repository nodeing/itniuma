大学英语四级速记单词库(1901-1957)

## 单词

### 1. considerable

```
adj. 相当大的；重要的，值得考虑的
```

```
n. 承包人；立契约者
```

```

```

n. 合作，协作； 协力

```
adj. 可行的；可能的；可实行的
```

```
n. 滤波器； 过滤器；筛选；滤光器

vt. 过滤；渗透；用过滤法除去

vi. 滤过；渗入；慢慢传开

```

```

```

vi. 伸展；足够买（或支付）
```
### 49. stroke

```

vt. （用笔等）画；轻抚；轻挪；敲击；划尾桨；划掉；（打字时）击打键盘

vi. 击球；作尾桨手，指挥划桨；（打字时）击打键盘
